# Папери Хорвата

Цей репозиторій є загальнодоступною версією форм для документів зроблених тим самим Лансом Морганом, з часом буде доповнюватись новими формами для різних відділів.
Головною перевагою подібного приступу є легке оновлення та зручність отримання форм в грі через можливість друкувати папери з .txt файлів. 
Все що стосується форматування та обсягу є під повною диктатурою ХорватБГ.
